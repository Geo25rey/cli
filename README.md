# Usage
<!-- usage -->
```sh-session
$ npm install -g mailscriptcli
$ mailscript COMMAND
running command...
$ mailscript (-v|--version|version)
mailscriptcli/0.1.0 darwin-x64 node-v14.15.0
$ mailscript --help [COMMAND]
USAGE
  $ mailscript COMMAND
...
```
